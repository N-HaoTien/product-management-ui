Product package
===============

This folder is the product UI package. The canonical components live in subfolders below:

- `./form` — Product form component (TS/HTML/CSS)
- `./list` — Product list component (TS/HTML/CSS)
- `./detail` — Product detail component

If you are editing components or templates, update the files under those subfolders. The root-level TS files in this folder are small re-exports kept for compatibility.
